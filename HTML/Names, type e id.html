Nomes e Tipos

name="nome", em tags de Input, Form, Option etc.. é obrigatório

para que a nosso site envie dados, e nossas outras páginas recebam

os dados e saibam de onde estão vindos.


Id e Type

Id ou Identificadores são utilizados para que nosso CSS (linguagem de marcação) ou Javascript

consiga saber onde desejamos fazer alterações e configurações

ID e Name são diferentes.

Utilizamos Name quando queremos identificar os dados adicionados

e ID para identificar tags
